# Scratch Archive

Completed experiments remain here when their evidence is still useful but they are no longer the active scratch task.

- `instagram/`: frozen 96-case Instagram corpus, acquisition record, completed loop plans/results, and canary runners. Tests still use representative cases from this corpus.
- `website-import/`: historical batch-import reports.
- `recipe-input/`: ignored generated output from the earlier recipe-input probe.

Archived plans preserve their original paths and commands as historical records. Use the current `scratch/README.md` for active entrypoints.
