Project: [[../Kitchen Sync|Kitchen Sync]]

# Purpose
Tracks planned purchases.

# Owns
- Shopping list items
- Checked/unchecked state
- Recipe source references
- Quantity multiplier application through the recipe engine
- Grouping similar ingredients where possible
- Manual free-text items

# Public API Questions
- What do I need to buy?
- Which recipe added this item?
- Has this item been purchased?
- Should this item move into the pantry?

# Open Questions
- How aggressive should ingredient grouping be?
- Should store/category ordering be part of v1?
